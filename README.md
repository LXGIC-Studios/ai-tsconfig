Generate optimal tsconfig.json for your project type.

## Install

```bash
npm install -g ai-tsconfig
```

## Usage

```bash
npx ai-tsconfig next
npx ai-tsconfig node
npx ai-tsconfig react
npx ai-tsconfig library
```

## Setup

```bash
export OPENAI_API_KEY=sk-...
```
