# ai-tsconfig

[![npm version](https://img.shields.io/npm/v/ai-tsconfig.svg)](https://www.npmjs.com/package/ai-tsconfig)
[![npm downloads](https://img.shields.io/npm/dm/ai-tsconfig.svg)](https://www.npmjs.com/package/ai-tsconfig)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)


Generate optimal tsconfig.json for your project type.

## Install

```bash
npm install -g ai-tsconfig
```

## Usage

```bash
npx ai-tsconfig next
npx ai-tsconfig node
npx ai-tsconfig react
npx ai-tsconfig library
```

## Setup

```bash
export OPENAI_API_KEY=sk-...
```
